# K NEWS Scraper
A Scraping CLI Application about Korean News Platforms

## How to use
```bash
npm install -g k-news-scraper

k-news-scraper -q [query]
```

### Options
- -q: query
- -a: search amount with date unit [default: 1]
- -u: search date unit [default: 'm']
- -d: start date [default: today]
